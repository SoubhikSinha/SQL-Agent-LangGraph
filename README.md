# **SQL Agent @ `LangGraph`**

This repository demonstrates a complete  **LLM-powered SQL Agent workflow**  using  **LangGraph**  to orchestrate tool-driven reasoning over a relational database. The project showcases how a Large Language Model can dynamically generate, validate, and execute SQL queries against a structured database using tools such as query generation, query validation, and database execution utilities. Built with  **LangChain’s SQLDatabase toolkit**,  **LangGraph’s stateful agent workflow**, and a  **Groq-hosted LLM**, the notebook illustrates how user questions can be translated into SQL, checked for correctness, executed on the database, and returned as structured responses.
<br>

The system follows a  **tool-augmented reasoning loop**, where the model iteratively plans actions, invokes tools, and refines queries before producing the final answer. By combining  **LangGraph’s deterministic workflow orchestration**with  **LangChain’s database abstractions**, the repository demonstrates how to build reliable, production-ready AI agents that can interact safely with structured data. The project serves as a practical, end-to-end reference for developing  **database-aware AI assistants**, highlighting best practices in agent design, SQL safety validation, and multi-step reasoning while emphasizing how graph-based execution improves control, observability, and reliability in LLM-powered applications.
